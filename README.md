# AI Code Reviewer (Proof of Concept)

Прототип пользовательского интерфейса и системы автоматизированного тестирования для интеграции AI-ассистента в процесс Code Review (проект KUMA).

## Реализованный функционал
- Минималистичный UI для отправки фрагментов кода.
- Mock-сервис, имитирующий ответы OpenAI API с базовым фильтром уязвимостей.
- Автоматизированное тестирование качества ответов AI (Jest).
- Интеграция статического анализатора (ESLint + eslint-plugin-security) для валидации сгенерированного кода.

## Стек технологий
- Frontend: HTML5, CSS3, Vanilla JS
- Тестирование: Node.js, Jest, ESLint (программный запуск)

## Инструкция по запуску

1. Установите зависимости:
   `npm install`

2. Запустите локальный сервер для просмотра интерфейса:
   `npx serve src`
   (Затем откройте `http://localhost:3000` в браузере)

3. Запустите автоматизированные тесты (проверка сценариев AI):
   `npm test`
